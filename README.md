# Larissa Scalzaretto — Portfólio 2026

Portfólio pessoal de Larissa Scalzaretto, Creative Designer com 7 anos de experiência em branding, UX/UI, web design e marketing digital.

## Acesse

🔗 [larissa-scalzaretto-designer.vercel.app](https://larissa-scalzaretto-designer.vercel.app/)

## Sobre o projeto

Site de portfólio desenvolvido com HTML, CSS e JavaScript puro — sem frameworks, sem dependências.

**Funcionalidades:**
- Tema claro/escuro com persistência no localStorage
- Design responsivo (mobile, tablet e desktop)
- Navbar fixa com menu hamburguer no mobile
- Efeitos de luz e glow nos botões
- Gradiente atmosférico no hero
- 8 cases com imagens e links externos
